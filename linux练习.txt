常用系统工作命令
-------------------------------------------------
echo 打印字符串，变量中的值
date 日期
timedatectl 编辑日期
reboot 重启系统
poweroff 关闭电脑
wget 获取web数据
ps 显示进程
pstree 显示进程树
top 显示进程负载信息
nice 用于调整进程优先级
pidof 查询某个进程pid码
kill 终止某个进程
killall 终止某个服务的全部进程

系统状态检测命令
-------------------------------------------------
ifconfig 获取网卡配置与网络状态等信息
uname 查看系统内核版本与系统架构等信息
uptime 查看系统的负载信息
free 显示当前系统中内存的使用量信息
who 查看当前登入主机的用户终端信息
last 调取主机的被访记录
ping 于测试主机之间的网络连通性
tracepath 显示数据包到达目的主机时途中经过的所有路由信息
netstat 显示如网络连接、路由表、接口状态等的网络相关信息
history 显示执行过的命令历史
sosreport 收集系统配置及架构信息并输出诊断文档

文件查找定位命令
-------------------------------------------------
pwd 显示用户当前所处的工作目录
cd 切换当前的工作路径
ls 显示目录中的文件信息
tree 以树状图的形式列出目录内容及结构
find 按照指定条件来查找文件所对应的位置
locate 用于按照名称快速搜索文件所对应的位置
whereis 按照名称快速搜索二进制程序（命令）、源代码以及帮助文件所对应的位置
which 按照指定名称快速搜索二进制程序（命令）所对应的位置

文本文件查看编辑命令
-------------------------------------------------
cat 查看纯文本文件（内容较少的）
more 命令用于查看纯文本文件（内容较多的）
head 命令用于查看纯文本文件的前 N 行
tail 查看纯文本文件的后 N 行或持续刷新文件的最新内容
tr 用于替换文本内容中的字符
wc 统计指定文本文件的行数
stat 查看文件的具体存储细节和时间等信息
grep 按行提取文本内容
cut 按“列”提取文本内容
diff 比较多个文件之间内容的差异
uniq 去除文本中连续的重复行
sort 对文本内容进行再排序

文件创建目录管理命令
-------------------------------------------------
touch 创建空白文件或设置文件的时间
mkdir 创建空白的目录
cp 复制文件或目录
mv 剪切或重命名文件
rm 于删除文件或目录
dd 按照指定大小和个数的数据块来复制文件或转换文件
file 查看文件的类型
tar 对文件进行打包压缩或解压
	[root@linuxprobe~]# tar czvf etc.tar.gz /etc
	-c 创建压缩文件
	-x 解开压缩文件
	-t 查看压缩包内有哪些文件
	-z 用 gzip 压缩或解压
	-j 用 bzip2 压缩或解压
	-v 显示压缩或解压的过程
	-f 目标文件名
	-p 保留原始的权限与属性
	-P 使用绝对路径来压缩
	-C 指定解压到的目录
